Utilizando Imagens no HTML

A sintaxe básica é: <img src="caminho_da_imagem" alt="texto alternativo">

Onde src é o atributo mais importante, define o caminho para a imagem.

E alt define um texto alternativo, para quando a imagem não carregar.

A tag img é auto-contida, não é necessário fecha-la.

A tag img possui vários atributos que podemos utilizar:

  • src: caminho para a imagem;
  • alt: texto alternativo;
  • width: largura da imagem;
  • height: altura da imagem;

src

Imagens Locais

Para imagens que estão no computador.

<img src="/minha_imagem.jpg">

<img src="/imagens/fotos/minha_imagem.jpg">

Imagens da Internet

Para exibir imagens que estão hospedadas na internet:

<img src="https://www.meusite.com/imagens/foto.jpg">

Texto alternativo

Permite difinir um texto alternativo, que será exibido caso a imagem não carregue.

Além disso, tambpem ajuda na acessibilidade, pois leitores de tela utilizados por deficientes visuais podem ler esse texto.

Definindo Tamanho

<img src="minha_imagem.jpg" width="100" height="200">>
