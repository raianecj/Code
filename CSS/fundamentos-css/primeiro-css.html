Primeiro código CSS

CSS inline

Adicionando o CSS inline para mudar a cor do body. É adicionado na própria tag que quer alterar.

<body style="background-color: red;">

CSS através da tag style

Adicionando a tag style no head para mudar a cor do título

<style> h1 { color: white; } <style>

CSS através da tag link para arquivo externo

Adicionando a tag link para incluir um arquivo CSS externo e criar o arquivo style.css

A tag link tem que ser adicionada no head.

<link rel="stylesheet" href="style.css">
